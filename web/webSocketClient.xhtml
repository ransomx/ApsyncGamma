WebSockets API

Tato stránka slouží ke zkoušení komunikace websockety. Budou zde uvedeny příkazy implementované na serveru, které lze použít použít pro komunikaci a server s nimi bude nakládat rozdílně.

Dále zde budou uvedena schémata pro JSON objekty, které využijeme pro jednotlivé typy komunikací.

Příkaz Popis
loginDevice() Přihlásí zařízení nacházející se pod daným číslem, které musí být v databázi - např. loginDevice(+420 775 761 325).
logoutDevice() odhlásí zařízení z aplikace pro konkrétně otevřenou session.
devMessages() Vypíše všechny kontakty a zprávy z komentálně přihlášeného zařízení.
devMessages(cislo) Vypíše všechny kontaktu daného tel. číslem. Příklad: devMessages(+420 876 874 541).
rsync Provede synchronizaci pro dané přihlášené číslo.
